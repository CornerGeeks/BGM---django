Data is by default escaped

{{ viewdata.html }}

Switch off auto escape: 
    {% templatetag openblock %} autoescape off {% templatetag closeblock %} {% templatetag openvariable %} viewdata.html {% templatetag closevariable %} {% templatetag openblock %} endautoescape

Auto escape off

{% autoescape off %}{{ viewdata.html }}{% endautoescape %}

Image in the static directory

Load at top of template 
    {% templatetag openblock %} load static {% templatetag closeblock %}
Actually showing the image 
    {% templatetag openblock %} static "nobody_50.png" {% templatetag closeblock %}
Same as 
 
    /static/nobody_50.png

    	There is a page named {% templatetag openvariable %} viewdata.page {% templatetag closevariable %}
    {% templatetag openblock %} else {% templatetag closeblock %}
    	There is a page
    {% templatetag openblock %} endif {% templatetag closeblock %}
    {% templatetag openblock %} for value in viewdata.array {% templatetag closeblock %}
    	Value: {% templatetag openvariable %} value {% templatetag closevariable %}
    {% templatetag openblock %} endfor {% templatetag closeblock %}
